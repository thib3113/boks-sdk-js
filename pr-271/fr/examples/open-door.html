Démo d'Ouverture de Porte ​

💡

Pour utiliser ces exemples interactifs, démarrez le Simulateur ou Connectez un véritable appareil à l'aide du tableau de bord au bas de votre écran.

Cette démo interactive permet de déverrouiller une Boks en utilisant un code PIN clavier valide (Code Maître ou Code Utilisateur).

⚠️ Non connecté. Veuillez utiliser le panneau de connexion en bas de la page.

Commande Ouvrir la Porte

Envoyer une commande d'ouverture standard à l'aide d'un code PIN à 6 chiffres.

Comment utiliser ​

  1. Connexion : Cliquez sur "Scanner & Connecter" et sélectionnez votre appareil.
  2. Code PIN : Saisissez le code PIN (6 caractères).
  3. Déverrouiller : Cliquez sur "Ouvrir la porte".

Indice

Si vous utilisez le simulateur, le code par défaut est 123456.
