Interactive Examples ​

Welcome to the interactive examples gallery. You can use these demos to test the SDK features either with a local hardware simulator or with your real Boks device.

💡

To use these interactive examples, start the Simulator or Connect a real device using the dashboard at the bottom of your screen.

Session Persistence

The connection state and simulated data are shared across all pages in this section. You only need to connect once.

Available Demos ​

  • Initialization: Set up a factory-fresh Boks with a new Master Key.
  • Open Door: Send a PIN code to unlock the device.
  • History Sync: Download and parse the internal event log.
  • Battery & Hardware: Read battery levels and detailed statistics.
