To use these interactive examples, start the Simulator or Connect a real device using the dashboard at the bottom of your screen.

CRITICAL WARNING

This software is currently theoretical and has not been tested on real hardware. Using this regeneration process will likely break compatibility with the official Boks app. Use at your own risk. This is reverse-engineered software.
