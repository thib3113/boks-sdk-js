Exemples Interactifs ​

Bienvenue dans la galerie d'exemples interactifs. Vous pouvez utiliser ces démos pour tester les fonctionnalités du SDK, soit avec un simulateur matériel local, soit avec votre véritable appareil Boks.

💡

Pour utiliser ces exemples interactifs, démarrez le Simulateur ou Connectez un véritable appareil à l'aide du tableau de bord au bas de votre écran.

Persistance de Session

L'état de la connexion et les données simulées sont partagés sur toutes les pages de cette section. Vous n'avez besoin de vous connecter qu'une seule fois.

Démos Disponibles ​

  • Initialisation : Configurez un Boks en sortie d'usine avec une nouvelle Clé Maître.
  • Régénération Seed : Remplacez la Clé Maître actuelle pour gérer les codes hors-ligne.
  • Ouverture de Porte : Envoyez un code PIN pour déverrouiller l'appareil.
  • Sync Historique : Téléchargez et analysez le journal d'événements interne.
  • Génération PIN Offline : Générez des codes d'accès localement via votre Clé Maître.
  • Santé Matérielle : Données de diagnostic en temps réel des piles et capteurs.
  • Gestion des Codes : Créez et gérez les codes clavier (Maître, Usage unique, etc.).
  • Gestion NFC : Enregistrez et supprimez des badges NFC (pour les Boks v4+).
  • Batterie & Matériel : Lisez les niveaux de batterie et les statistiques détaillées.
  • Visualiseur de Paquets : Décodez et visualisez des paquets bruts du protocole Boks.
